Visual Studio Code

Текстовый редактор для программистов.

Достоинства:

  • развитые инструменты для написания программного кода;
  • «умная» подстановка языковых конструкций;
  • интеграция с исполняющими средами (Google Chrome, PHP, Node.js и др.);
  • встроенный полнофункциональный отладчик;
  • большое количество расширений;
  • удобные средства для установки расширений и управления ими;
  • широкие возможности настройки (включая темы оформления).

Недостатки:

  • не очень удобные клавиатурные комбинации;
  • довольно большой объем;
  • не очень высокое быстродействие на больших файлах.

Atom

Текстовый редактор для программистов.

Достоинства:

  • поддержка практически всех языков, используемых в промышленном программировании;
  • удобные клавиатурные комбинации;
  • огромное количество расширений;
  • удобные средства для установки расширений и управления ими;
  • широкие возможности настройки (включая темы оформления).

Недостатки:

  • некоторые полезные инструменты изначально отсутствуют (исправляется расширениями);
  • очень большой объем;
  • низкое быстродействие на больших файлах.

Notepad++

Текстовый редактор для программистов.

Достоинства:

  • поддержка многих языков, используемых в промышленном программировании;
  • удобные клавиатурные комбинации;
  • компактность;
  • исключительное быстродействие.

Недостатки:

  • крайне бедный набор инструментов;
  • небольшое количество расширений.
